# Testing & Development

개발 기본 루틴과 품질 게이트를 정리한 페이지다.

## 자주 쓰는 명령

```bash
npm test
npm run lint
npm run build
npm run dev
```

## 현재 상태 요약

- 테스트 통과
- 빌드 통과
- 현재 워킹 트리 기준 린트 실패 2건
